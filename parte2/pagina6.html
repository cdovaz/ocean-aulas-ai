🐝 O que é um Swarm?

Um swarm de agentes é um grupo de agentes especializados que colaboram para resolver uma tarefa complexa. Cada agente tem um papel, ferramentas e objetivo definidos.

  • Divisão de responsabilidades
  • Comunicação entre agentes
  • Orquestração de tarefas

⚡ Quando usar

Swarms são úteis quando uma única LLM não consegue cobrir bem todas as habilidades necessárias ou quando o problema se divide naturalmente em etapas especializadas.

  • Relatórios complexos (pesquisa + escrita + revisão)
  • Automação de workflows multi-etapas
  • Code review, testes e documentação
