Agente de Relatórios na Prática

Exemplo simplificado de implementação com LangGraph

🧩 Estados do grafo

  • receber_pedido: valida entradas do usuário
  • buscar_dados: consulta Supabase e APIs
  • recuperar_contexto: busca no Qdrant
  • gerar_relatorio: chama Gemini API
  • revisar: checa formato e fontes
  • entregar: salva e retorna ao usuário

📝 Prompt para o Gemini

"""Você é um analista de dados. Gere um relatório sobre {tema} no período {periodo}. Use apenas as fontes fornecidas abaixo. Formato: markdown com resumo, insights e recomendações. Fontes: {contexto_qdrant} {dados_supabase}"""

🐍 Estrutura do código (conceitual)

from langgraph.graph import StateGraph builder = StateGraph(AgentState) builder.add_node("receber_pedido", receber_pedido) builder.add_node("buscar_dados", buscar_dados) builder.add_node("recuperar_contexto", recuperar_contexto) builder.add_node("gerar_relatorio", gerar_relatorio) builder.add_node("revisar", revisar) builder.add_node("entregar", entregar) builder.set_entry_point("receber_pedido") builder.add_edge("receber_pedido", "buscar_dados") builder.add_edge("buscar_dados", "recuperar_contexto") # ... demais arestas e condições graph = builder.compile()
