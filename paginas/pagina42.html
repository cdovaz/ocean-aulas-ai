🕸️ Grafo LangGraph

Modela o RAG como um grafo de estado, permitindo loops e decisões condicionais.

pergunta
→
retrieve
→
generate
→
END
class RAGState(TypedDict): pergunta: str contexto: List[Document] resposta: str workflow = StateGraph(RAGState) workflow.add_node("retrieve", retrieve) workflow.add_node("generate", generate) workflow.set_entry_point("retrieve") workflow.add_edge("retrieve", "generate") app = workflow.compile()

🚀 FastAPI

  • Endpoint POST /perguntar recebe a pergunta
  • Invoca o grafo compilado rag_app.invoke()
  • Retorna pergunta e resposta em JSON
  • GET /health para health check
@app.post("/perguntar") def perguntar(request: PerguntaRequest): resultado = rag_app.invoke({ "pergunta": request.pergunta }) return {"resposta": resultado["resposta"]}
