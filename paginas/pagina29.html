Quando usar Fine-tuning?

Escolha a técnica certa para cada problema

Situação Solução recomendada
Modelo erra formato ou estilo específico Fine-tuning
Precisa de conhecimento factual atualizado RAG + prompt engineering
Tarefa muito específica e repetitiva Fine-tuning
Poucos exemplos disponíveis Few-shot prompting
Precisa corrigir alucinações em fatos RAG
Quer reduzir tamanho do prompt Fine-tuning

📌 Regra de ouro: fine-tuning ensina como responder, não o que responder. Para conhecimento novo, use RAG.
