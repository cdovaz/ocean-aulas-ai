Do PDF ao vector database em 5 passos

1Carregar PDF
2Chunking
3Embeddings
4Conectar Qdrant
5Indexar vetores

⚙️ Setup

  • Instale: langchain, langchain-google-genai, langchain-qdrant, qdrant-client, pypdf, python-dotenv
  • Crie arquivo .env com GOOGLE_API_KEY, QDRANT_URL e QDRANT_COLLECTION_NAME
  • Suba o Qdrant via Docker Compose

📐 Configuração

Modelo de embeddings gera vetores de 768 dimensões. Ajuste o VectorParams conforme o modelo.

# 1. Carregar e dividir documento loader = PyPDFLoader("documento.pdf") docs = loader.load() splitter = RecursiveCharacterTextSplitter( chunk_size=1000, chunk_overlap=200 ) chunks = splitter.split_documents(docs) # 2. Embeddings Gemini embeddings = GoogleGenerativeAIEmbeddings( model="models/text-embedding-004" ) # 3. Criar coleção e indexar client.create_collection( collection_name="rag_documents", vectors_config=VectorParams(size=768, distance=Distance.COSINE) ) vector_store.add_documents(chunks)
