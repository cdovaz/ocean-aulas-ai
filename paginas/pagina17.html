Tecnologias para Criar Agentes

Ferramentas no-code, low-code e frameworks para construir agentes de IA

Z

Zapier

Plataforma no-code para automação de workflows. Permite criar agentes simples conectando apps e serviços sem escrever código.

No-code Low-code
Exemplo: Agente que lê e-mails, extrai dados e atualiza planilhas automaticamente.
n8n

n8n

Ferramenta open source de automação de workflows com suporte a nodes de IA. Oferece mais controle que plataformas no-code e pode ser self-hosted.

Open Source Low-code
Exemplo: Agente que integra CRM, envia mensagens no Slack e consulta APIs externas.
C

CrewAI

Framework Python para criar equipes de agentes colaborativos. Cada agente tem um papel, objetivo e ferramentas específicas.

Framework Python Open Source
Exemplo: Equipe com pesquisador, redator e revisor que cria um artigo completo.
LG

LangGraph / LangChain

LangChain conecta LLMs a ferramentas e dados. LangGraph adiciona grafos de estado para construir agentes complexos com ciclos e ramificações.

Framework Open Source
Exemplo: Agente de suporte que decide entre buscar FAQ, consultar API ou escalar para humano.
